# **App Name**: WA Invitation Sender

## Core Features:

- Add Invitation: Allow users to add invitation details including 'Invitation Name' and 'Digital Invitation Link'.
- Invitation List: Display a list of added invitations with their names and links.
- Send Invitation via WhatsApp: Enable sending invitation messages via WhatsApp with a pre-defined message format that includes the invitation name and digital invitation link.

## Style Guidelines:

- Primary color: Soft purple (#A69CAC) to convey a sense of sophistication and modernity suitable for event invitations.
- Background color: Very light gray (#F0F0F0), a desaturated version of the primary color, creating a clean and unobtrusive backdrop.
- Accent color: Dusty rose (#D0B4DE), analogous to the primary hue, for highlighting interactive elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif to give the application a blend of modern aesthetic with a touch of warmth and approachability.
- Use simple, outlined icons to represent actions like 'add', 'send', and 'edit'.
- A clean, single-column layout optimized for mobile viewing with clear sections for adding invitations, viewing the invitation list, and sending invitations.
- Subtle transitions and animations when adding, deleting, or sending invitations to provide a smooth user experience.
